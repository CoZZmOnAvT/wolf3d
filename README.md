# wolf3d

### Controls:
 - WASD - Movement
 - SPACE - Shoot
 - ENTER - Open Command Line
 - TAB - Enter Full Screen Mode
 - ESC - EXIT
 
##### Avalible commands:
 - raisethesun - Heals you
 - godmode - gives you immortality
 - goto [level_number] - sends you to choosen level
 - displaymap - shows map of the level
 - exit - EXIT
